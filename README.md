**Lumina** is a cosmic-themed, interactive Tarot and Astrology web application. 
Featuring a dynamic starfield background and a magical UI, it provides an immersive reading experience. Its primary goal is to **generate perfectly optimized prompts** that users can directly copy and paste into generative AI models (like ChatGPT or Gemini) for in-depth, personalized Tarot and Natal Chart readings.

---

## ✨ Features

- **🔮 Tarot Readings**: Draw from a full 78-card deck with interactive UI and auto-generate AI prompts for your reading.
- **🔭 Western Astrology**: Calculate your Natal Chart (Sun, Moon, Rising) and get tailored AI prompts for deep insights.
- **🦋 Immersive UI**: Fully animated falling starfield and glassmorphism interface, guided by our mascot, Lumi.

---

## 🛠 Tech Stack

- **Frontend Core**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS v4
- **Animation Components**: Framer Motion (3D Card Flips, Layout Transitions, Floating Elements)
- **Icons**: Lucide React
- **Routing**: React Router DOM (Single Page Application)

## 🚀 Getting Started

To run the project locally on your machine, follow these steps:

```bash
# 1. Install dependencies
npm install

# 2. Run the local development server
npm run dev

# 3. Open your browser and navigate to 'http://localhost:5173/'
```
